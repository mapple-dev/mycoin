### Wow How to make it

	sudo apt-get install build-essential \
                         libssl-dev \
                         libdb5.1++-dev \
                         libboost-all-dev \
                         libqrencode-dev \
                         libminiupnpc-dev

	cd src/
	make -f makefile.unix USE_UPNP=-
	./mycoind 

### The Wallet-QT

	qmake
	make
